Website Credits

This website was built using the following technologies:

  • SvelteKit - Was used to build the website. Combined with the static adapter, made it so much easier to build pages and create a page for every MIDI fast and easy. I love SvelteKit
  • TailwindCSS - Was used to style the website. I used TailwindCSS for their quick utility classes and the fact it reduces the css to a minimal level.
  • DaisyUI - Was used to quickly create beautiful UI elements such as the buttons, badges, navbar, footer, and more. The light and dark themes were also created by DaisyUI.
  • change-theme - Was used to change the theme using the button on the top right, and saves your current theme to local storage allowing you to keep your theme when you revisit the website.
  • Github Pages - Was used to host the website. Github pages is a free service that allows you to host static websites for free. I used the static adapter for SvelteKit to build the website and then used Github pages to host it. Github is awesome
Check out the source code for this website on Github!
